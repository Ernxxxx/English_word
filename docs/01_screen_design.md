            ┌──────────────┐    ┌──────────────┐

## 画面詳細

### 1. オンボーディング（3画面）

```
┌────────────────────────┐
│                        │
│      [イラスト]        │
│                        │
│   レベル別に学習       │
│   基礎・標準・上級     │
│   自分に合った         │
│   レベルから始めよう   │
│                        │
│       ● ○ ○           │
│                        │
│     [ 次へ ]           │
└────────────────────────┘

画面1: レベル別学習の説明
画面2: カード学習の操作説明（タップで反転、スワイプで評価）
画面3: 毎日続けてstreak！→ [始める]ボタン
```

## コンポーネント一覧

| コンポーネント名 | 用途 |
|-----------------|------|
| LevelCard | ホームのレベルカード |
| FlashCard | 学習画面の単語カード |
| ProgressBar | 進捗バー |
| EvaluationButtons | 覚えた/まだ/あとでボタン |
| WordListItem | 単語一覧の行 |
| WordEditForm | 単語追加/編集フォーム |
| StreakBadge | 連続日数バッジ |
| PremiumBanner | Premium誘導バナー |
| OnboardingPage | オンボーディング1ページ |
