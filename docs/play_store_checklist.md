# Google Play Store 公開前チェックリスト

## 概要

このチェックリストは、Google Play Storeにアプリを公開するために必要な設定・準備事項をまとめたものです。

---

## 1. アプリ署名設定

### 1.1 署名キーの作成

- [ ] **キーストアファイルの作成**

  ```bash
  keytool -genkey -v -keystore english-word-release.keystore \
    -alias english-word-key \
    -keyalg RSA -keysize 2048 -validity 10000
  ```

- [ ] **キーストア情報の記録**
  - キーストアパスワード
  - キーエイリアス
  - キーパスワード
  - 有効期限

- [ ] **キーストアファイルの安全な保管**
  - バックアップを複数箇所に保存
  - パスワードマネージャーで管理
  - **重要**: 紛失すると同じパッケージ名でアップデートできなくなる

### 1.2 Google Play アプリ署名

- [ ] **Play App Signing（推奨）に登録**
  - Google Play Consoleで「アプリの署名」を有効化
  - アップロード鍵とアプリ署名鍵の分離

- [ ] **build.gradleに署名設定を追加**
  ```kotlin
  signingConfigs {
      create("release") {
          storeFile = file("path/to/keystore")
          storePassword = System.getenv("KEYSTORE_PASSWORD")
          keyAlias = "english-word-key"
          keyPassword = System.getenv("KEY_PASSWORD")
      }
  }
  buildTypes {
      release {
          signingConfig = signingConfigs.getByName("release")
      }
  }
  ```

## 2. リリーストラック選択

### 2.1 トラック種類と用途

| トラック         | 用途                       | 対象者       |
| ---------------- | -------------------------- | ------------ |
| 内部テスト       | 開発チームのテスト         | 最大100人    |
| クローズドテスト | 限定ユーザーのベータテスト | 招待制       |

- [ ] **Step 2: クローズドテスト（2-4週間）**

  - より広いユーザーからのフィードバック
  - パフォーマンステスト

  - 段階的ロールアウト推奨（10% → 50% → 100%）

## 3. コンテンツレーティング

### 3.1 IARC質問票の回答準備

- [ ] **暴力表現**: なし
- [ ] **性的表現**: なし
- [ ] **薬物・アルコール**: なし
- [ ] **ギャンブル**: なし
- [ ] **不快な言葉**: なし
- [ ] **ユーザー間インタラクション**: なし
- [ ] **位置情報共有**: なし
- [ ] **個人情報収集**: なし（または最小限）

### 3.2 予想されるレーティング

- **日本（CERO）**: A（全年齢対象）
- **北米（ESRB）**: E（Everyone）
- **欧州（PEGI）**: 3

### 3.3 質問票回答手順

1. [ ] Play Console → 「アプリのコンテンツ」 → 「コンテンツのレーティング」
2. [ ] 「質問票を開始」をクリック
3. [ ] カテゴリ「教育」を選択
4. [ ] 質問に正直に回答
5. [ ] レーティング結果を確認・承認

## 6. 定期購入（サブスクリプション）設定

### 6.1 Play Console での設定

- [ ] **収益化 → 商品 → 定期購入**に移動
- [ ] **新規作成**

### 6.2 Premium プランの設定

| 項目           | 設定値                         |
| -------------- | ------------------------------ |
| 商品ID         | `premium_monthly`              |
| 名前           | Premium（月額）                |
| 説明           | 広告なし、レベル・単語数無制限 |
| デフォルト価格 | ¥300 / 月                      |
| 請求期間       | 1か月                          |
| 無料試用期間   | なし（または7日間）            |
| 猶予期間       | 3日間                          |
| アカウント保留 | 有効（30日間）                 |

### 6.3 特典の設定

- [ ] **特典リスト**
  - 広告非表示
  - レベル数無制限
  - 単語数無制限
  - 学習統計グラフ

### 6.4 テスト用ライセンス

- [ ] **テスターアカウントを登録**
  - Play Console → 「設定」 → 「ライセンステスト」
  - テスト用Googleアカウントを追加
  - テスト購入が無料になる

## 7. AdMob 設定

### 7.1 AdMobアカウント設定

- [ ] **AdMobアカウント作成**（未作成の場合）
- [ ] **アプリを登録**
  - アプリ名: 英単語帳 - レベル別学習
  - プラットフォーム: Android

### 7.2 広告ユニットの作成

| 広告タイプ           | 用途           | 広告ユニットID       |
| -------------------- | -------------- | -------------------- |
| バナー広告           | ホーム画面下部 | ca-app-pub-XXXX/YYYY |
| インタースティシャル | 学習完了時     | ca-app-pub-XXXX/ZZZZ |

### 7.3 app-ads.txt の設定

- [ ] **app-ads.txt をウェブサイトに配置**
  - `https://yourdomain.com/app-ads.txt`
  - AdMobで生成されたコードを貼り付け

### 7.4 AdMob App ID の設定

- [ ] **AndroidManifest.xml に追加**
  ```xml
  <meta-data
      android:name="com.google.android.gms.ads.APPLICATION_ID"
      android:value="ca-app-pub-XXXXXXXXXXXXXXXX~YYYYYYYYYY"/>
  ```

### 7.5 Play Console での確認

- [ ] **AdMobとPlay Consoleをリンク**
  - AdMob → 「設定」 → 「リンクされたアプリ」
  - Play Console のアプリと紐付け
