■ 主な機能

【レベル別学習】
基礎・標準・上級の3つのレベルから、自分に合ったレベルを選んで学習できます。初心者から上級者まで、段階的にステップアップ！

【フラッシュカード形式】
カードをタップして英単語と日本語訳を確認。シンプルな操作で直感的に学習できます。

【日本語訳・例文付き】
すべての単語に日本語訳と例文（英語・日本語）を収録。文脈の中で自然に単語を覚えられます。

【科学的な復習システム（SRS）】
スペースド・リピティション・システム（間隔反復学習法）を採用。覚えた単語は間隔を空けて、まだの単語は繰り返し出題されます。

【3段階評価】
・覚えた → 次回の復習間隔が延長
・まだ → すぐに再出題
・あとで → 同じ間隔で復習

【連続学習記録（ストリーク）】
毎日の学習を続けると連続日数がカウントアップ！モチベーションを維持して習慣化をサポートします。

【進捗管理】
各レベルの習得率をグラフで確認。どれだけ覚えたか一目でわかります。

【自分だけの単語帳】
自分で単語を追加・編集可能。オリジナルの単語帳を作成できます。

【オフライン対応】
インターネット接続なしで利用可能。いつでもどこでも学習できます。

## 競合との差別化ポイント

1. **シンプルな操作性** - 複雑な機能を排除し、学習に集中できる設計
2. **レベル別分類** - 自分のレベルに合わせた学習が可能
3. **日本語例文付き** - 英語例文だけでなく日本語訳も収録
4. **科学的な復習システム** - SRSで効率的な記憶定着
5. **連続学習記録** - ゲーミフィケーションでモチベーション維持
6. **オフライン対応** - 通信環境を気にせず学習可能
7. **広告控えめ** - 学習体験を損なわない広告配置
